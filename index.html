Steps to set up github

  1. Set up git on a browser
    • Search for gitbash on google
    • click on git download
    • choose windows 64bit
    • click on setup after download
    • click yes
    • keep clicking next
    • change default editor to vscode
    • click the first checkbox before clicking install
    • click launch git bash
  2. The terminal has been launched
  3. Create a project
    • mkdir (name of project)
    • change directory cd (name of project should allign with above)
    • type "code ." in the environment and click enter to launch in vscode
    • vscode environment should open
  5. In VScode
    • click yes i trust this author
    • create new file index.html
    • create html structure
  7. To see your terminal
    • click on 3 dots, click terminal
    • click on new terminal
    • beside powershell click the drop down and pick git bash
    • the name should match
  9. To set up your terminal
    • type git init and enter
    • U will show beside our code meaning it has not been tracked
    • git add space . (to add git to our code to track it)
    • git commit -m"(write a description e.g tutorial) click enter. It will say author identity unknown
    • type git config --global user.email "your email" and enter
    • type git config --global user.name "your username" and enter
    • commit git again and "A" should have disappeared because git is already tracking
    • you should see your files and insertions
  11. If an addition has been made
    • git commit -m"I have added a paragraph"
  13. To see all you've done
    • type git log and you will see username, email, date, time
  15. To create a github account
    • search for github on your browser
    • click on create account
    • enter your details
    • solve the puzzle to verify your account
    • A code will be sent to your email
  17. Create a repository
    • click on create repository / new
    • click on public
    • add a readMe
    • create repository
  19. how to push vscode to repository
    1. type git push --set-upstream
    2. to get your link go to git hub click code, and copy the https link
    3. give a space and paste the link i.e git push --set-upstream https://pomole...
    4. you will be redirected to an authorization page when it is the first time
    5. click on sign in with browser
  21. how to comfirm your push
    1. type git push in vscode
    2. back in git hub, click settings, branches
    3. click code and you will see your uploaded code
  23. Test
